# login-php
simple login with Docker file

# For docker:
- Open the terminal
- git clone https://github.com/Lpdw2017/login-php.git
- cd login-php
- docker-compose build
- docker-compose up -d
- docker ps (For display the ports)

# For display the login page 
- /login.php

# For the database
-Import (login.sql) in your database
